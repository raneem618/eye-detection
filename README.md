Face Detection Camera Alarm

Real-time face detection with eye monitoring and alarm alert.

Features

Detects faces and monitors eyes using Google ML Kit.

Triggers an alarm if both eyes are closed for more than 1.5 seconds.

Draws rectangles around faces on the camera preview.

Automatic stop/reset of alarm when eyes open.

Tech

Language: Kotlin

Platform: Android

Dependencies: Google ML Kit Face Detection

Usage

Open in Android Studio.

Add your alarm sound in res/raw/ (e.g., alarm_sound.mp3).

Run the app on an Android device.

The alarm will play if eyes stay closed too long.
